Simple plugin to show a message in woocommerce checkout page

### Description 

this plugin can show a message in the Woocommerce chechout page with full compatibility 
with the old checkout page and the new chechout block with the use of the shortcode
```php
[woocommerce_custom_features]
```

### Installation 

1. Make sure you have Woocommerce plugin installed and active
2. upload and install .zip
3. activate plugin
4. if the Woocommerce plugin is older than 8.3 then theres nothing else left to do
but if its newer then you can either delete the checkout block and use the old shortcode [woocommerce_checkout] 
or add this plugin shortcode ([woocommerce_custom_features]) to the checkout page
